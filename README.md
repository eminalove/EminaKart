# KonradKart
A Mario Kart clone written in C++ in my own engine.

This was the project I used for my college coursework project. It is just like Mario Kart in that you can play with up to 4 other players using split screen, and an additional set of computer controlled racers.

The racing style is unique in that there is no global gravity; you conform to the track much like in F-Zero X or similar games. This allows for complex twisting tracks that wouldn't be possible otherwise.

The game is completely written in my own engine in C++, and uses a few libraries for linking OpenGL and for audio.

Player 1 uses the keyboard and the rest of the players use controllers, although this can be tweaked in the main class and on release I plan to change this.
